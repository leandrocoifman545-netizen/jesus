## 7. Resumen ejecutivo

### Los numeros que importan

| Metrica | Valor |
|---------|-------|
| CLR promedio CON CTA | 10.30% |
| CLR promedio SIN CTA | 1.85% |
| Multiplicador CTA | 5.56x |
| Sweet spot (promedio) | 30-45s + CTA |
| Sweet spot (mediana) | 30-45s + CTA |
| Keyword mas efectiva | RETO (21.58% CLR, n=36) |
| Videos analizados | 220 |

### 5 insights accionables

1. **CTA multiplica comentarios 5.56x.** No es sutil — es la diferencia entre 2% y 10% CLR. Para ADP, SIEMPRE poner CTA keyword.

2. **30-45s es el sweet spot para CTA por promedio, pero 60-90s es el mas robusto** (n=96 videos, CLR 11.59%). Para ADP: apuntar a 60-75s con CTA.

3. **"RETO" arrasa como keyword** porque activa FOMO + comunidad + documentacion publica. Equivalente ADP: "comenta TALLER" o "comenta PLAN" con contexto de reto/transformacion.

4. **Videos sin CTA tienen 2-5x mas views** pero no convierten. Usar para alcance (awareness) y videos con CTA para conversion. No mezclar: o pedis accion o vas por reach.

5. **Los outliers tienen pocas views pero ratio comments/likes altisimo.** Son audiencia caliente respondiendo a un CTA especifico. Mejor 1,500 views con 300% CLR que 100K views con 0.3% CLR para inbound.
