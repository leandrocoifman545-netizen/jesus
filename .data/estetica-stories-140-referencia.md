## SPECS DE REPRODUCCION PARA SOFTWARE

### Spec 1: Progressive Reveal
```
base_image: 1 foto (no cambia)
layers: [
  { slide: 1, elements: [{ type: "text", content: "pregunta", position: "top" }] },
  { slide: 2, elements: [{ type: "badge", number: 1, color: "#FF0000" }, { type: "arrow", from: "badge", to: "right" }, { type: "text_bubble", content: "punto 1" }] },
  { slide: 3, elements: [prev_elements + nuevo badge + arrow + text] },
  { slide: 4, elements: [prev_elements + nuevo badge + arrow + text] }
]
max_slides: 4
rule: cada slide hereda TODO lo anterior + agrega 1 elemento
```

### Spec 2: Film Grain Nocturno
```
filter: {
  grain: { amount: 25-40, size: 1.5-2.0 },
  exposure: -0.5 to -1.0 EV,
  contrast: +10-15%,
  warmth: -5 to -10 (ligeramente frio),
  saturation: -15 to -25%
}
text: {
  font: Inter/Helvetica/SF Pro,
  color: #FFFFFF,
  accent_color: #FF8C00 (naranja/dorado),
  weight: {title: 700, body: 400},
  underline: wavy, color accent, 1 keyword max per slide
}
negative_space: min 40% del frame vacio
```

### Spec 3: Fondo Blur con Texto
```
background: {
  image: foto lifestyle,
  filter: gaussian_blur(radius: 15-25px),
  overlay: rgba(0,0,0, 0.1-0.2)
}
text: {
  position: center_vertical,
  alignment: center,
  font_size: { h1: 32-40px, h2: 20-24px },
  color: #FFFFFF,
  highlight: { type: "background_fill", color: accent, padding: 4px 8px }
}
```

### Spec 4: Screenshot Embebido
```
screenshot: {
  border_radius: 12-16px,
  shadow: { x: 0, y: 4, blur: 12, color: rgba(0,0,0,0.3) },
  max_width: 70% del frame,
  position: center o offset 20% de centro,
  max_per_slide: 2
}
context_text: {
  position: above or below screenshot,
  font_size: 16-20px,
  content: 1-2 lineas max explicando que es el screenshot
}
```

### Spec 5: Foto con Bordes Redondeados
```
photo: {
  border_radius: 16-20px,
  shadow: { x: 0, y: 4, blur: 16, color: rgba(0,0,0,0.25) },

  rotation: 0-3deg (opcional, para fotos antiguas)

background: blur o negro solido
arrangement: {
  single: centrada,
  dual: side_by_side con 8-12px gap,
  triple: triangulo o horizontal
}
```

## REGLAS DE DISEÑO ACTUALIZADAS (v2)

1. **1 idea por slide** — nunca saturar (excepto collages de prueba intencionalmente densos)
2. **Jerarquia tipografica clara:** H1 bold grande → H2 regular → Body chico → CTA bold
3. **Maximo 1-2 highlights de color por slide** — el resto blanco/gris
4. **Foto siempre tiene proposito:** comunica lifestyle/resultado/vulnerabilidad/proceso
5. **Screenshots = credibilidad** — bordes redondeados, sombra, integrados en composicion
6. **Espacio negativo min 30%** — el frame no se llena entero
7. **Paleta consistente por secuencia** — 1 accent color en toda la secuencia
8. **CTA siempre keyword DM** — ninguno de los 8 creadores usa link en bio
9. **Progresion visual:** secuencias van de oscuro/vulnerable → luminoso/aspiracional
10. **Branding sutil:** logo solo en CTA final o esquina, nunca en cada slide
11. **Progressive reveal = retencion** — si la misma foto aparece, el ojo busca que cambio
12. **Zonas muertas:** nada importante en 0-80px top ni 1700-1920px bottom
13. **Fotos antiguas siempre con bordes redondeados** — nunca fullbleed
14. **Color accent en keywords, no en frases enteras** — 1-3 palabras por slide
15. **Cifras en grande y en color** — los numeros son anclas visuales ($23k, $2M, 10K/mes)
