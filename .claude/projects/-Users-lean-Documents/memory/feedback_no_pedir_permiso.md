description: Nunca pedir permiso para hacer algo. Mostrar el trabajo y preguntar opinión.
type: feedback
---

No pedir permiso para ejecutar. Mostrar las cosas antes de hacer y preguntar "qué opinás", no "querés que lo haga?".

**Why:** Lean quiere velocidad y opinión, no burocracia. Pedir permiso frena el flujo.
**How to apply:** Ante cualquier cambio propuesto, mostrar el draft/plan concreto y preguntar opinión. Nunca "¿querés que...?" sino "esto es lo que haría, ¿qué opinás?".
